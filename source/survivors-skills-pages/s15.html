Сам себе доктор

Описание

У Вас появляется способность лечить себя без аптечки, но со скоростью 50% от обычной скорости лечения. Повышает эффективность лечащих предметов на {процентов}.

Особенности

Не требует перезарядки, работает на протяжении всего матча Самолечение доступно только при получении обычной травмы В процессе самолечения могут появляться проверки реакции Провал одной из них спящим выжившим может вернуть его из мира снов Прогресс самолечения с помощью навыка является накопительным и не сбрасывается, если прервать процесс по каким-либо причинам При наличии аптечки игра предложит выбор: использовать аптечку (М2) или навык (М1) для самолечения Если заряды аптечки будут израсходованы, то процесс самолечения продолжится с эффективностью навыка Чем выше уровень навыка, тем меньше скорость траты аптечек при самолечении Полное самолечение с помощью навыка 3-го уровня занимает ~32 секунды Каждое отличное прохождение проверки реакции уменьшает время, необходимое для полного самолечения, на 1 сек. (но уменьшает итоговое кол-во очков крови, получаемое игроком при полном исцелении, вер. 1.9.4+) Совместное использование с навыком "Познания в ботанике" ускоряет самолечение до ~24 сек.

Примечания

Необходим для получения достижения "Опытная Клодетт".
